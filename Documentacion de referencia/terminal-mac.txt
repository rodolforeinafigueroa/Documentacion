Comandos generales:
cd + nombre de la carpeta: Para ingresar a la carpeta.
cd + .. devolverse al directorio superior.
mkdir: crear una nueva carpeta.
touch: crear un nuevo archivo.
rm + nombre del archivo: Eliminar un archivo.
rm -rf + nombre de la carpeta: Eliminar una carpeta.
ls: ver los archivos y carpetas que se encuentran dentro del directorio que estamos trabajando.
ls -a: incluye los archivos ocultos.
vim + nombre del archivo: permite editar el archivo
mv nombreviejo nombre nuevo: cambiar el nombre de una archivo

## Consideraciones vim
para salir de vim hay que pulsar la tecla escape, despues escribir dos puntos ":", seguido de la letra q si queremos salir o w si queremos guardar los cambios.

Ejemplo para salir y guardar los cambios:
Tecla escape seguido de :wq
